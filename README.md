A videoplayer(no audio) and image viewer for show things in terminal using ascii extended character

### Build Requirements
* ncursesw5
* opencv>=3

### Build
From the directory 

```
make
```

### Control

* 'q' to quit
* 'p' to pause
* 'w' to forward 2s
* 's' to backward 2s

forward, backward skip causes exit in case of image render
